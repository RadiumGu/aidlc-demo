# 组件定义

## 组件概览

```
+--------------------------------------------------+
|                   AWSomeShop                      |
|                                                   |
|  +------------+                                   |
|  | Frontend   |  React SPA                        |
|  +------+-----+                                   |
|         |                                         |
|  +------v-----------+                             |
|  | Gateway Service  |  API 网关 + 认证拦截         |
|  +--+---+---+---+--+                             |
|     |   |   |   |                                 |
|  +--v-+ | +-v-+ +-v--+                           |
|  |Auth| | |Pts| |Ord |                           |
|  +----+ | +---+ +----+                           |
|         |                                         |
|      +--v-----+                                   |
|      |Product |                                   |
|      +--------+                                   |
+--------------------------------------------------+
```

---

## 组件 1: Auth Service（认证服务）

**职责**: 用户身份管理、认证与授权

| 能力 | 描述 |
|------|------|
| 用户管理 | 用户 CRUD、角色管理 |
| 认证 | 登录验证、JWT 签发 |
| 令牌管理 | Token 验证、刷新、失效 |
| 安全防护 | 登录失败限制、密码加密 |

**接口提供方**: Gateway Service（Token 验证）、Frontend（登录/登出）

---

## 组件 2: Product Service（商品服务）

**职责**: 商品信息全生命周期管理

| 能力 | 描述 |
|------|------|
| 商品 CRUD | 创建、编辑、删除商品 |
| 商品查询 | 列表（分页+筛选+搜索）、详情 |
| 库存管理 | 库存扣减、恢复 |
| 上下架 | 商品状态管理 |

**接口提供方**: Frontend（浏览/管理）、Order Service（库存扣减）

---

## 组件 3: Points Service（积分服务）

**职责**: 员工积分账户管理

| 能力 | 描述 |
|------|------|
| 账户管理 | 积分账户初始化、余额查询 |
| 积分变动 | 发放、扣除、兑换扣减 |
| 流水记录 | 积分变动历史查询 |

**接口提供方**: Frontend（查询/管理）、Order Service（兑换扣减）

---

## 组件 4: Order Service（订单服务）

**职责**: 兑换订单全流程管理

| 能力 | 描述 |
|------|------|
| 订单创建 | 积分兑换下单（协调积分+库存事务） |
| 订单查询 | 员工订单、管理员全部订单 |
| 状态流转 | 待处理→待发货→已发货→已完成/已取消 |
| 订单取消 | 取消并退还积分+恢复库存 |

**依赖**: Product Service（库存）、Points Service（积分）

---

## 组件 5: Gateway Service（网关服务）

**职责**: 统一入口、认证拦截、路由分发

| 能力 | 描述 |
|------|------|
| 认证拦截 | JWT Token 验证（已实现） |
| 路由转发 | 按路径分发到后端服务（已实现） |
| 操作者注入 | X-Operator-Id 头注入（已实现） |
| 角色权限 | 管理员接口访问控制（需新增） |

**依赖**: Auth Service（Token 验证）

---

## 组件 6: Frontend（前端）

**职责**: 员工端和管理端用户界面

| 能力 | 描述 |
|------|------|
| 员工端 | 登录、商品浏览/搜索/筛选、兑换、订单/积分查看 |
| 管理端 | 商品管理、积分发放/扣除、订单管理 |
| 状态管理 | 认证状态、全局应用状态 |
| API 对接 | 通过 Gateway 调用所有后端 API |
