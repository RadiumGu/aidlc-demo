# 业务概览

## 业务上下文

AWSomeShop 是一个内部员工福利电商平台，通过积分兑换系统实现员工福利发放与消费。

```
+--------------------------------------------------+
|                  AWSomeShop 系统                  |
|                                                   |
|  +----------+    +----------+    +----------+     |
|  |   员工   |--->|  前端UI  |--->|  网关    |     |
|  | (浏览/   |    | (React)  |    | (Gateway)|     |
|  |  兑换)   |    +----------+    +-----+----+     |
|  +----------+                          |          |
|                                        v          |
|  +----------+    +----------+    +----------+     |
|  |  管理员  |--->|  前端UI  |--->| 认证服务 |     |
|  | (配置/   |    | (Admin)  |    |  (Auth)  |     |
|  |  管理)   |    +----------+    +----------+     |
|  +----------+                          |          |
|                    +-------------------+          |
|                    v         v         v          |
|              +--------+ +--------+ +--------+    |
|              | 商品   | | 积分   | | 订单   |    |
|              | 服务   | | 服务   | | 服务   |    |
|              +--------+ +--------+ +--------+    |
+--------------------------------------------------+
```

## 业务描述

AWSomeShop 是一个最小可行产品（MVP），旨在验证员工积分兑换系统的商业模式。系统允许员工使用"AWSome积分"浏览和兑换预选产品，管理员可以手动配置产品信息和管理员工积分。

## 业务事务

| 业务事务 | 描述 |
|---------|------|
| **员工登录** | 员工通过用户名密码登录系统，获取 JWT Token |
| **商品浏览** | 员工查看可用商品列表，包括积分价格、库存等信息 |
| **积分兑换** | 员工使用积分兑换商品，生成订单并扣减积分（待实现） |
| **积分查询** | 员工查看当前积分余额和兑换历史（待实现） |
| **商品管理** | 管理员创建、编辑商品信息（名称、价格、库存等） |
| **积分管理** | 管理员手动发放或扣除员工积分（待实现） |
| **订单管理** | 管理员查看和管理兑换订单（待实现） |

## 业务术语

| 术语 | 含义 |
|------|------|
| **AWSome积分** | 员工福利积分，用于兑换商品 |
| **积分价格 (pointsPrice)** | 商品兑换所需积分数量 |
| **市场价 (marketPrice)** | 商品的市场参考价格 |
| **SKU** | 商品库存单位编号 |
| **操作者ID (operatorId)** | 经认证的用户标识，通过网关注入请求头 |

## 各组件业务职责

### frontend（前端）
- **职责**: 提供员工和管理员的 Web 操作界面
- **核心功能**: 登录、商品浏览、积分展示、管理后台

### gateway-service（网关服务）
- **职责**: 统一入口，路由分发，认证拦截
- **核心功能**: JWT 令牌验证、请求路由、操作者ID注入

### auth-service（认证服务）
- **职责**: 用户认证与授权
- **核心功能**: 登录验证、JWT 令牌签发与验证

### product-service（商品服务）
- **职责**: 商品信息管理
- **核心功能**: 商品 CRUD、商品列表查询

### points-service（积分服务）
- **职责**: 积分账户管理
- **核心功能**: 积分查询、积分发放与扣除（待实现）

### order-service（订单服务）
- **职责**: 兑换订单管理
- **核心功能**: 订单创建、订单查询（待实现）
