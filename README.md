# aidlc-demo

AWS + AI 开发训练营项目代码仓库

## 目录结构

- `frontend/` — 前端（awsome-shop-frontend）
- `gateway-service/` — 网关服务
- `auth-service/` — 认证服务
- `product-service/` — 商品服务
- `points-service/` — 积分服务
- `order-service/` — 订单服务
- `plan/` — 设计稿

## 分支说明

每个子目录均有对应的 `baseline`（基础骨架）和 `intermediate`（部分实现）分支可参考。
